# Tenure Tech

**Tenure Tech** is a modern multi-page business website designed for a premium laptop subscription platform. It offers flexible hardware subscription plans for students, professionals, creators, and startups while shifting costs from **CapEx to OpEx**.  

🔗 **Live Demo:** https://gowtham-mummidi.github.io/Tenure_Tech/security.html

---

##  Features

-  Clean and modern responsive UI  
-  Multi-page website structure  
-  Laptop subscription pricing tiers  
-  Startup fleet solutions page  
-  Security moat / asset protection section  
-  Mobile-friendly navigation menu  
-  Smooth animations and hover effects  
-  Toast notifications with JavaScript  
-  Fleet calculator for business users  

---

##  Tech Stack

- **HTML5** – Structure  
- **CSS3** – Styling & Responsive Design  
- **JavaScript (Vanilla JS)** – Interactivity  

---

##  Pages Included

- **Home Page** – Landing page with hero section  
- **Subscriptions** – Pricing plans & laptop tiers  
- **For Startups** – Fleet subscriptions with calculator  
- **Security Moat** – Device protection & hardware security
